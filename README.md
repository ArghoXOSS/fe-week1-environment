# FE Week 1 Environment

## Overview

This repository contains my FlyRank AI Front-End Engineering internship Week 1 environment setup.

## Tech Stack

- Node.js
- Git
- Cursor IDE
- JavaScript

## Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [Git](https://git-scm.com/)

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/ArghoXOSS/fe-week1-environment.git
cd fe-week1-environment
npm install
```
